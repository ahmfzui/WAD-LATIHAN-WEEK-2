CREDIT RISK ANALYSIS

Project Objective: The aim of this project is to improve credit risk assessment accuracy for a multifinance company. By utilizing historical borrower data and machine learning algorithms, the project develops a predictive model to effectively forecast credit risk. This enables financial institutions to make informed lending decisions, optimizing portfolio profitability while minimizing the risk of borrower default.

SENTIMEN ANALYSIS: CYBERBULLYING

Project Objective: The aim of this project is to develop a robust cyberbullying detection system for social media platforms, specifically targeting Indonesian online interactions. Using machine learning algorithms and a dataset of TikTok comments, the project seeks to classify comments as bullying or non-bullying. This system will help raise awareness, prevent online harassment, and protect users by providing real-time sentiment analysis, allowing stakeholders to take appropriate action against harmful content, and fostering safer online environments.

HYDROPONIC SMART SYSTEM (AI & IoT)

Project Objective: The aim of this project is to enhance environmental sustainability through the development of HITOSH (Hydroponic Tech House). By utilizing IoT technology and machine learning algorithms, HITOSH enables real-time monitoring and control of hydroponic systems. This project seeks to optimize plant growth by managing key environmental factors such as temperature, humidity, pH, and nutrient levels, ensuring optimal plant health. The solution empowers hydroponic farmers and agribusinesses to make data-driven decisions, improving crop yields while minimizing resource usage and environmental impact.
